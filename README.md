
# 🐍 Snake Water Gun Game

## 📌 Description

Snake Water Gun ek simple Python game hai jo classic "Rock Paper Scissors" jaisa concept follow karta hai.
Is game me user aur computer ke beech match hota hai jisme har round me winner decide hota hai.

---

## 🎮 Game Rules

* 🐍 Snake drinks Water → Snake wins
* 💧 Water douses Gun → Water wins
* 🔫 Gun kills Snake → Gun wins

---

## 🚀 Features

* User vs Computer gameplay
* Random choice generation (computer side)
* Simple and interactive CLI interface
* Beginner-friendly Python project

---

## 🛠️ Technologies Used

* Python
* Random Module
